MySQL கமெண்ட்கள்

கமெண்ட்கள் SQL கூற்றுகளின் பகுதிகளை விளக்க அல்லது SQL கூற்றுகளின் செயல்பாட்டைத் தடுக்க பயன்படுகின்றன.

ஒற்றை வரி கமெண்ட்கள்

ஒற்றை வரி கமெண்ட்கள் -- உடன் தொடங்கும்.

-- மற்றும் வரியின் முடிவுக்கு இடையே உள்ள எந்த உரையும் புறக்கணிக்கப்படும் (செயல்படுத்தப்படாது).

பின்வரும் எடுத்துக்காட்டு ஒரு ஒற்றை வரி கமெண்டை விளக்கமாகப் பயன்படுத்துகிறது:

-- Select all:
SELECT * FROM Customers;

பின்வரும் எடுத்துக்காட்டு ஒரு வரியின் முடிவைப் புறக்கணிக்க ஒற்றை வரி கமெண்டைப் பயன்படுத்துகிறது:

SELECT * FROM Customers -- WHERE City='Berlin';

பின்வரும் எடுத்துக்காட்டு ஒரு கூற்றைப் புறக்கணிக்க ஒற்றை வரி கமெண்டைப் பயன்படுத்துகிறது:

-- SELECT * FROM Customers;
SELECT * FROM Products;

பல வரி கமெண்ட்கள்

பல வரி கமெண்ட்கள் /* உடன் தொடங்கி */ உடன் முடிகின்றன.

/* மற்றும் */ இடையே உள்ள எந்த உரையும் புறக்கணிக்கப்படும்.

பின்வரும் எடுத்துக்காட்டு ஒரு பல வரி கமெண்டை விளக்கமாகப் பயன்படுத்துகிறது:

/*Select all the columns
of all the records
in the Customers table:*/
SELECT * FROM Customers;

பின்வரும் எடுத்துக்காட்டு பல கூற்றுகளைப் புறக்கணிக்க பல வரி கமெண்டைப் பயன்படுத்துகிறது:

/*SELECT * FROM Customers;
SELECT * FROM Products;
SELECT * FROM Orders;
SELECT * FROM Categories;*/
SELECT * FROM Suppliers;

ஒரு கூற்றின் ஒரு பகுதியை மட்டும் புறக்கணிக்க, /* */ கமெண்டைப் பயன்படுத்தவும்.

பின்வரும் எடுத்துக்காட்டு ஒரு வரியின் ஒரு பகுதியைப் புறக்கணிக்க கமெண்டைப் பயன்படுத்துகிறது:

SELECT CustomerName, /*City,*/ Country FROM Customers;

பின்வரும் எடுத்துக்காட்டு ஒரு கூற்றின் ஒரு பகுதியைப் புறக்கணிக்க கமெண்டைப் பயன்படுத்துகிறது:

SELECT * FROM Customers WHERE (CustomerName LIKE 'L%'
OR CustomerName LIKE 'R%' /*OR CustomerName LIKE 'S%'
OR CustomerName LIKE 'T%'*/ OR CustomerName LIKE 'W%')
AND Country='USA'
ORDER BY CustomerName;

WHERE City = 'Berlin';
