MySQL DROP TABLE Statement

MySQL இல் DROP TABLE அறிக்கையைப் பயன்படுத்தி தரவுத்தளத்தில் இருக்கும் அட்டவணையை நீக்க கற்றுக்கொள்ளுங்கள்

MySQL DROP TABLE அறிக்கை

DROP TABLE அறிக்கை ஒரு தரவுத்தளத்தில் இருக்கும் அட்டவணையை நீக்க பயன்படுகிறது.

தொடரியல்

DROP TABLE table_name;

⚠️ குறிப்பு:

ஒரு அட்டவணையை நீக்குவதற்கு முன் கவனமாக இருங்கள். ஒரு அட்டவணையை நீக்குவது அட்டவணையில் சேமிக்கப்பட்ட முழு தகவலையும் இழப்பதற்கு வழிவகுக்கும்!

MySQL DROP TABLE எடுத்துக்காட்டு

பின்வரும் SQL கூற்று இருக்கும் "Shippers" அட்டவணையை நீக்குகிறது:

DROP TABLE Shippers;

MySQL TRUNCATE TABLE

TRUNCATE TABLE அறிக்கை ஒரு அட்டவணையின் உள்ளே உள்ள தரவை நீக்க பயன்படுகிறது, ஆனால் அட்டவணையை அல்ல.

தொடரியல்

TRUNCATE TABLE table_name;
