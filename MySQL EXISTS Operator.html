MySQL EXISTS ஆபரேட்டர்

EXISTS ஆபரேட்டர் ஒரு சப்குவரியில் எந்தவொரு பதிவின் இருப்பையும் சோதிக்கப் பயன்படுகிறது.

சப்குவரி ஒன்று அல்லது அதற்கு மேற்பட்ட பதிவுகளை வழங்கினால் EXISTS ஆபரேட்டர் TRUE ஐ வழங்குகிறது.

EXISTS தொடரியல்

SELECT column_name(s)
FROM table_name
WHERE EXISTS
(SELECT column_name FROM table_name WHERE condition);

MySQL EXISTS எடுத்துக்காட்டுகள்

பின்வரும் SQL கூற்று TRUE ஐ வழங்குகிறது மற்றும் 20 க்கும் குறைவான தயாரிப்பு விலை கொண்ட சப்ளையர்களை பட்டியலிடுகிறது:

SELECT SupplierName
FROM Suppliers
WHERE EXISTS (SELECT ProductName FROM Products WHERE Products.SupplierID = Suppliers.supplierID AND Price < 20);

பின்வரும் SQL கூற்று TRUE ஐ வழங்குகிறது மற்றும் 22க்கு சமமான தயாரிப்பு விலை கொண்ட சப்ளையர்களை பட்டியலிடுகிறது:

SELECT SupplierName
FROM Suppliers
WHERE EXISTS (SELECT ProductName FROM Products WHERE Products.SupplierID = Suppliers.supplierID AND Price = 22);
