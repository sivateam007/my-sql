MySQL MIN() and MAX() Functions

MySQL MIN() மற்றும் MAX() சார்புகள்

MySQL MIN() மற்றும் MAX() சார்புகள்

MIN() சார்பு தேர்ந்தெடுக்கப்பட்ட நெடுவரிசையின் சிறிய மதிப்பைத் திருப்பித் தருகிறது.

MAX() சார்பு தேர்ந்தெடுக்கப்பட்ட நெடுவரிசையின் பெரிய மதிப்பைத் திருப்பித் தருகிறது.

MIN() தொடரமைப்பு

SELECT MIN(column_name)
FROM table_name
WHERE condition;

MAX() தொடரமைப்பு

SELECT MAX(column_name)
FROM table_name
WHERE condition;

MIN() எடுத்துக்காட்டு

பின்வரும் SQL அறிக்கை மலிவான தயாரிப்பின் விலையைக் கண்டறியும்:

எடுத்துக்காட்டு: உங்கள் சொந்த SQL சர்வரைப் பெறுங்கள்

SELECT MIN(Price) AS SmallestPrice
FROM Products;

MAX() எடுத்துக்காட்டு

பின்வரும் SQL அறிக்கை மிகவும் விலையுயர்ந்த தயாரிப்பின் விலையைக் கண்டறியும்:

எடுத்துக்காட்டு

SELECT MAX(Price) AS LargestPrice
FROM Products;

பயிற்சி

❓ பின்வருவனவற்றில் எது சரியான SQL அறிக்கையாகும்:

SELECT Price(MAX) FROM Products;
✗ தவறு! இது தவறான தொடரமைப்பு. MAX() சார்பு நெடுவரிசைப் பெயருக்கு முன் வர வேண்டும்
SELECT * FROM Products MAX(Price);
✗ தவறு! இது தவறான தொடரமைப்பு. MAX() சார்பு SELECT பகுதியில் மட்டுமே பயன்படுத்தப்பட முடியும்
SELECT MAX(Price) FROM Products;
✓ சரி! இது சரியான SQL தொடரமைப்பாகும் - MAX() சார்பு SELECT பகுதியில் சரியாகப் பயன்படுத்தப்பட்டுள்ளது
