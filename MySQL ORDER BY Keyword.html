MySQL ORDER BY முக்கிய வார்த்தை

ORDER BY முக்கிய வார்த்தை முடிவு-தொகுப்பை ஏறுவரிசை அல்லது இறங்குவரிசையில் வரிசைப்படுத்த பயன்படுகிறது.

ORDER BY முக்கிய வார்த்தை இயல்பாக ஏறுவரிசையில் பதிவுகளை வரிசைப்படுத்துகிறது. பதிவுகளை இறங்குவரிசையில் வரிசைப்படுத்த, DESC முக்கிய வார்த்தையைப் பயன்படுத்தவும்.

ORDER BY தொடரமைப்பு

SELECT column1, column2, ...
FROM table_name
ORDER BY column1, column2, ... ASC|DESC;

பின்வரும் SQL அறிக்கை "Customers" அட்டவணையில் இருந்து அனைத்து வாடிக்கையாளர்களையும் தேர்ந்தெடுக்கிறது, "Country" மற்றும் "CustomerName" நெடுவரிசைகளால் வரிசைப்படுத்தப்பட்டது. இதன் பொருள் இது Country மூலம் வரிசைப்படுத்துகிறது, ஆனால் சில வரிசைகள் ஒரே Country ஐக் கொண்டிருந்தால், அவற்றை CustomerName மூலம் வரிசைப்படுத்துகிறது:

எடுத்துக்காட்டு

SELECT * FROM Customers
ORDER BY Country, CustomerName;

எடுத்துக்காட்டு

SELECT * FROM Customers
ORDER BY Country ASC, CustomerName DESC;
