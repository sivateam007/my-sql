RDBMS என்றால் என்ன?

RDBMS என்பது Relational Database Management System என்பதன் சுருக்கமாகும்.

RDBMS என்பது ஒரு தொடர்புடைய தரவுத்தளத்தை பராமரிக்க பயன்படும் ஒரு நிரலாகும்.

RDBMS என்பது MySQL, Microsoft SQL Server, Oracle, மற்றும் Microsoft Access போன்ற அனைத்து நவீன தரவுத்தள அமைப்புகளுக்கும் அடிப்படையாகும்.

RDBMS தரவுத்தளத்தில் உள்ள தரவை அணுக SQL கோரிக்கைகளைப் பயன்படுத்துகிறது.

தரவுத்தள அட்டவணை என்றால் என்ன?

ஒரு அட்டவணை என்பது தொடர்புடைய தரவு உள்ளீடுகளின் தொகுப்பாகும், மேலும் இது நெடுவரிசைகள் மற்றும் வரிசைகளைக் கொண்டுள்ளது.

ஒரு நெடுவரிசை அட்டவணையில் உள்ள ஒவ்வொரு பதிவு பற்றிய குறிப்பிட்ட தகவல்களை வைத்திருக்கிறது.

ஒரு பதிவு (அல்லது வரிசை) என்பது ஒரு அட்டவணையில் இருக்கும் ஒவ்வொரு தனிப்பட்ட உள்ளீடாகும்.

தொடர்புடைய தரவுத்தளம் என்றால் என்ன?

ஒரு தொடர்புடைய தரவுத்தளம் அட்டவணைகளின் வடிவத்தில் தரவுத்தள உறவுகளை வரையறுக்கிறது. அட்டவணைகள் ஒவ்வொன்றுக்கும் பொதுவான தரவின் அடிப்படையில் ஒன்றோடொன்று தொடர்புடையவை.
