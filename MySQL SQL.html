MySQL SQL

MySQL SQL மொழி

SQL என்றால் என்ன?

SQL என்பது தொடர்புடைய தரவுத்தளங்களைக் கையாளுவதற்கான நிலையான மொழியாகும்.

SQL தரவுத்தள பதிவுகளைச் செருக, தேட, புதுப்பிக்க மற்றும் நீக்க பயன்படுத்தப்படுகிறது.

SQL ஐ எவ்வாறு பயன்படுத்துவது

பின்வரும் SQL அறிக்கை "Customers" அட்டவணையில் உள்ள அனைத்து பதிவுகளையும் தேர்ந்தெடுக்கிறது:

எடுத்துக்காட்டு: உங்கள் சொந்த SQL சர்வரைப் பெறுங்கள்

SELECT * FROM Customers;

மனதில் கொள்ள வேண்டியது...

SQL முக்கிய வார்த்தைகள் case sensitive அல்ல: select என்பது SELECT போன்றதே

இந்த டுடோரியலில் நாம் அனைத்து SQL முக்கிய வார்த்தைகளையும் upper-case இல் எழுதுவோம்.

SQL அறிக்கைகளுக்குப் பிறகு அரைப்புள்ளி?

சில தரவுத்தள அமைப்புகள் ஒவ்வொரு SQL அறிக்கையின் முடிவிலும் ஒரு அரைப்புள்ளி தேவைப்படுகிறது.

சேவையகத்திற்கு அதே அழைப்பில் ஒன்றுக்கு மேற்பட்ட SQL அறிக்கையை இயக்க அனுமதிக்கும் தரவுத்தள அமைப்புகளில் ஒவ்வொரு SQL அறிக்கையையும் பிரிக்க நிலையான வழி அரைப்புள்ளியாகும்.

இந்த டுடோரியலில், நாம் ஒவ்வொரு SQL அறிக்கையின் முடிவிலும் அரைப்புள்ளியைப் பயன்படுத்துவோம்.

மிக முக்கியமான SQL கட்டளைகளில் சில

SELECT

ஒரு தரவுத்தளத்திலிருந்து தரவைப் பிரித்தெடுக்கிறது

UPDATE

ஒரு தரவுத்தளத்தில் தரவைப் புதுப்பிக்கிறது

DELETE

ஒரு தரவுத்தளத்திலிருந்து தரவை நீக்குகிறது

INSERT INTO

தரவுத்தளத்தில் புதிய தரவைச் செருகுகிறது

CREATE DATABASE

ஒரு புதிய தரவுத்தளத்தை உருவாக்குகிறது

ALTER DATABASE

ஒரு தரவுத்தளத்தை மாற்றுகிறது

CREATE TABLE

ஒரு புதிய அட்டவணையை உருவாக்குகிறது

ALTER TABLE

ஒரு அட்டவணையை மாற்றுகிறது

DROP TABLE

ஒரு அட்டவணையை நீக்குகிறது

CREATE INDEX

ஒரு குறியீட்டை உருவாக்குகிறது (தேடல் விசை)

DROP INDEX

ஒரு குறியீட்டை நீக்குகிறது

பயிற்சி

❓ பின்வருவனவற்றில் எது ஒரு பொதுவான SQL அறிக்கையாகும்?

Customers(ID, CustomerName, City);
✗ தவறு! இது சரியான SQL தொடரமைப்பு அல்ல
SELECT * FROM Customers;
✓ சரி! இது ஒரு பொதுவான SQL SELECT அறிக்கையாகும்
{table: 'Customers', fields: 'ID CustomerName, City'}
✗ தவறு! இது JSON வடிவமைப்பு, SQL அறிக்கை அல்ல
